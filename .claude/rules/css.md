# CSS / Styling Rules

- Use Tailwind CSS utility classes for styling, not custom CSS in `index.css`
- Keep `index.css` minimal: only Tailwind directives and essential global styles
- Prefer Tailwind's built-in utilities over custom CSS classes
- Use Tailwind's theme configuration in `tailwind.config.js` for custom values
